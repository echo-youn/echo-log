# 자바스크립트로 탭메뉴 만들기
UI를 구성 할때 종종 탭 메뉴를 구현해야 할 때가 있습니다.

탭 메뉴는 단순한 웹 페이지를 웹 어플리케이션으로 탈바꿈 해주기 딱 좋은 기술입니다.

사용자는 어플리케이션을 사용하고 있는 듯한 느낌을 받게 해줍니다.

어떻게 만드는지 알아봅시다.

예시는 JQuery를 사용하여 구현되었습니다.
